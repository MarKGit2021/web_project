Как работает наш сайт?

Общая информация

Добро пожаловать на сайт "название сайта!!!"! Здесь вы можете посмотреть информацию по вашему запросу. Наш сайт предоставляет вам информацию, добавленную и проверенную нашими пользователями.

Возврат на главную страницу

Чтобы вернуться на главную страницу достаточно нажать на название сайта в верхнем левом углу

Регистрация

Чтобы зарегистрироваться на нашем сайте, нажмите кнопку "Зарегистрироваться" в правом верхнем углу. Вам откроется форма, где вам нужно будет указать информацию о себе. После этого, на сайте будет создана ваша учетная запись. Чтобы воспользоваться ею, войдите на сайт (с помощью кнопки "Войти" в правом верхнем углу. После регистрации вам будет доступна возможность оценивать предоставляемую информацию, получать информацию для с помощью нашего API, а также добавлять новую. Для неавторизованных пользователей информация будет доступна только для чтения.

Добавление новой информации

Чтобы добавить новую информацию, нажмите на кнопку "Добавить" в правом верхнем углу и заполните форму. Можно загрузить .txt и .html файлы. При загрузке файла и добавлении информации в форму будет учитываться информация в форме

Информацию можно загружать только зарегистрированным пользователям

Личный кабинет

Чтобы перейти в личный кабинет, нажмите на кнопку с вашим именем в правом верхнем углу. В личном кабинете вам доступен список добавленной вами информации и ваш личный API-токен, который вы можете сменить.

REST-API (для разработчиков)

Запрос всей информации по слову

Чтобы получить всю доступную информацию по слову вам нужно отправить GET-запрос на "/api/all_information/word", где word - это слово, по которому вы ищете информацию. ВАЖНО: в headers не забудте указать свой токен!

Запрос основной информации по слову

Чтобы получить основную информацию по слову вам нужно отправить GET-запрос на "/api/main_information/word", где word - это слово, по которому вы ищете информацию. ВАЖНО: в headers не забудте указать свой токен!

Добавление информации

Чтобы добавить информацию вам нужно отправить POST-запрос на "/api/information" и указать в теле запроса параметры: text - содержимое информации (тип данных - строка) word - основное слово для поиска (тип данных - строка) words - список слов для поиска (тип данных - строка или null, формат: слово; слово). ВАЖНО: в headers не забудте указать свой токен!
